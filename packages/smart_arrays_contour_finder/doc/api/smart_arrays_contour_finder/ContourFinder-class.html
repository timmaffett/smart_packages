ContourFinder class

Straightforward method of contouring some surface represented as a regular triangular mesh aiding in visualizing three dimensional surfaces on a two dimensional medium (e.g.computer graphics screen). Applications: Contouring functions of two variables such as topological maps, weather maps, spectrocopic data. For details of the algorithm see http://paulbourke.net/papers/conrec or https://debrief.github.io/tutorial/contouring_algorithm.html.

findContour(List<Float64List> d, int ilb, int iub, int jlb, int jub, Float64List x, Float64List y, int nc, Float64List z) → void
findContour is a contouring subroutine for rectangularily spaced data It emits calls to a line drawing function, see ContourRenderer, which draws a contour map. The coordinates emitted are in the same units given in the x andy arrays. Any number of contour levels may be specified but they must be in order of increasing value.
