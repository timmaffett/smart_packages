DSetStore class

The purpose of the class is to provide an interface for storing DSet dataset objects (i.e. 1D or 2D array with metadata) in the browser. A concrete implementation is DSetDB which stores datasets in an "IndexedDB" database. It could also be used for a concrete implementation using the html5 file system HTML5 FS or other. This class provides methods to:

  • save data or components of the data to the storage device
  • save data or components of the data from the storage device
  • Delete, copy, rename stored data
  • Group stored data into categories ("projects") and manage projects
  • Save / read information common to all stored data ("global properties") Usage: README.md and example.dart of this package.

dscompDel(DSKey dskey, String dsc) → Future
Deletes the component dsc of the dataset stored under dskey.
dscompLoad(DSKey dskey, String dsc) → Future<List<Float64List>>
Inverse of dscompSave: Returns matrix stored under dskey as the dataset component dsc. Returns null if the component doesn't exist, or any error occured.
dscompSave(DSKey dskey, List<Float64List> matrix, String dsc) → Future
Saves matrix as the dataset component dsc, e.g. as DSC_REAL, under the identifier DSKey. A p otentially existing dsc component will be overridden. Throws a message upon error. dsc is one of DSC_REAL, DSC_IMAG, DSC_REAL, DSC_COMPR_POS, DSC_COMPR_NEG, DSC_IMAG_IR, DSC_IMAG_RI, DSC_IMAG_ROW, DSC_IMAG_COL.
dsCopy(DSKey sourceDskey, DSKey destDskey) → Future<bool>
Makes a copy of the dataset identified by sourceDskey and stores it under the new identifier destDskey. Note: for 2D data, if array components other than DSC_REAL and DSC_IMAG are present, these are not copied (TODO). Returns false if error. Also throws error message.
dsDel(List<DSKey> dskeys, bool imagOnly) → Future<bool>
Deletes all datasets with the identifiers dskeys. If imagOnly is true only the DSC_IMAG component of a daatset is deleted. Returns true on success.
dsExist(DSKey datasetKey) → Future<bool>
Returns true if a dataset with datasetKey exists in the store.
dsimagLoad(DSKey dskey) → Future<List<Float64List>>
Convenience method to load the component DSC_IMAG of a dataset defined by dskey from this data store. Returns null if no such component there, or on error. For 1D datasets, the result has only 1 List entry which is the DSC_IMAG component. For 2D, the result is the DSC_IMAG matrix.
dskeysWithProject(String projectName) → List<DSKey>
Returns all DSKeys belonging to projectName. The result is compiled from all stored datasets managed by this DSetStore. The result can be empty, but not null.
dsLoad(DSKey dskey) → Future<DSet>
Loads and returns the dataset identified by dskey from this data store. The resulting dataset will contain the metadata and the component DSetStore.DSC_REAL. For performance reason no other components, should they if present (e.g. DSC_IMAG), are loaded because the respective arrays might be large. They have to be loaded, when needed, separately using dscompLoad. Throws an error message in case of a problem.
dsmetaLoad(DSKey dskey) → Future<Map<String, String>>
Returns the metadata of a dataset stored under dskey. Returns null on error or no metadata. The error message is then available in load_meta_error.
dsmetaSave(DSKey dskey, DSet dset) → Future<bool>
Saves the metadata meta of the dset under dskey.
dsRen(DSKey sourceDatasetKey, DSKey destDatasetKey) → Future<bool>
Renames a dataset identified by sourceDatasetKey to the new identifier destDatasetKey.
dsSave(DSKey dskey, DSet ds) → Future
Saves the dataset ds under the identifier dskey in this store. If ds already exists, it is overriden silently. Throws an error message in case of a problem.
gpropDel(String propName) → Future<bool>
Deletes the property propName from the PropStore common ("global") to all datasets of this DSetStore. Returns true if succeeded.
gpropGet(String propName) → String
Returns the value of the property propName from the propStore maintained by this DSetStore.
gpropSave(String propName, String propValue) → Future<String>
Saves the property with name propName and with value propValue to the PropStore common ("global") to all datasets of this DSetStore. Returns propName in the case of success, null if failed. The latter is true if the db could not be opened or created.
projectDel(String projectname) → Future<bool>
Deletes all datasets managed by this DSetStore that belong to the project projectName.
projectExists(String project) → bool
Returns true if project exists in this DSetStore.
projectRen(String projectName, String newName) → Future<bool>
Renames project projectName to newName. This means that all datasets belonging to projectName will belong to newName afterwards. Returns false if newName==projectName.
projectsGet() → Map<String, String>
Returns all available project names in the key list of this Map. They are compiled from all datasets managed by this DSetStore (can be done because each dataset MUST belong to a project). The Map values (other than the keys are not currently used in this package.
