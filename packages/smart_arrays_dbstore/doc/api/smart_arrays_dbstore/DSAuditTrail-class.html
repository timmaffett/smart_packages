DSAuditTrail class

Provides functionalities for maintaining an audit trail for a dataset of type DSet. The purpose of an audit trail is to store information about the history of a dataset, e.g.: who changed the dataset at which time by applying which procedure for which reason? Details are up to the user of this class. This class just provides methods to maintain and store an audit trail. An audit trail will be stored together with the metadata of a dataset under the key DSET_AUDIT_TRAIL in jsonified form. Usage: call addToAuditTrail.
