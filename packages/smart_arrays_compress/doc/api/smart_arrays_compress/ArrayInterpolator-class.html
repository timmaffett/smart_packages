ArrayInterpolator class

Expands an array to a larger size by inserting extra points whose values are computed from their neighbours by linear interpolation of the array data.

interpolateArray(Float64List data, int newSize) → Float64List
Makes an array data larger by linear interpolation. newSize is the desired size after interpolation. Returns a new array of newSize. Works evens if newSize is smaller than data size, but this is not a true compression preserving minima an maxima. See CompressedArray1D instead.
linearInterpolate(double before, double after, double atPoint) → double
Interpolates two values before, after linearly at the position atPoint which must be in 0, 1 and returns the result.
