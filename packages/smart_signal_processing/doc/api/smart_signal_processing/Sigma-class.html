Static Methods

meanValue(Float64List array, int ixstart int ixend) → double
Returns the mean value of the numbers in array in the region ixstart, ixend (ixend exclusive) ixstart and ixend may be null, equivalent to the left/rightmost index.
sigma(Float64List array, int ixstart int ixend) → double
Returns the standard deviation of the numbers in array in the region ixstart, ixend (ixend exclusive): sigma = Sqrt(variance(arr)) ixstart and ixend may be null, equivalent to the left/rightmost index.
sigma2D(List<Float64List> matrix, int posneg, double ymax) → double
variance(Float64List array, int ixstart int ixend) → double
Returns the variance of the numbers in array in the region ixstart, ixend (ixend exclusive): variance = mean(x**2) - (mean(x))**2 ixstart and ixend may be null, equivalent to the left/rightmost index.
