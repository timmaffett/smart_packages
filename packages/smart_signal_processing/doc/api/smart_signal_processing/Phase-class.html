Phase class

magnitude(Float64List reals, Float64List imags, bool sqrt) → void
Applies in-place magnitude or power calculation to the complex- valued array whith the real part reals and the imaginary part imags. according to the following equations: realsi := realsi**2 + imagsi**2 if sqrt is false ("power") realsi := sqrt(realsi**2 + imagsi**2) if sqrt is true ("magnitude") [...]
phase(Float64List reals, Float64List imags, double a0, double a1, bool inverse) → void
Applies in-place phasing (rotation in the complex plane) to the complex- valued array whith the real part reals and the imaginary part imags. according to the following equations: realsi := realsi*cos(phii) - imagsi*sin(phii) imagsi := imagsi*cos(phii) + realsi*sin(phii) with phii := a0 + (i/(reals.length-1))*a1, i=0...reals.length-1 [...]
