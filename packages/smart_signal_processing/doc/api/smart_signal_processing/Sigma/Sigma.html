sigma method

double sigma (Float64List array, int ixstart int ixend)

Returns the standard deviation of the numbers in array in the region ixstart, ixend (ixend exclusive): sigma = Sqrt(variance(arr)) ixstart and ixend may be null, equivalent to the left/rightmost index.

Implementation

static double sigma(Float64List array, int ixstart, int ixend) {
  if (ixstart == null) ixstart = 0;
  if (ixend == null) ixend = array.length - 1;
  return math.sqrt(variance(array, ixstart, ixend));
}
