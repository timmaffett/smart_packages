Numerics class

This class provides a number of useful numeric calculations with vectors and matrices, such as multiplication, transpose, invert. Some of the provided methods are based on the respective js MIT-licensed code of https://github.com/sloisel/numeric.

compareMM(List<List<double>> mat1, List<List<double>> mat2, double epsilon) → String
Compares two matrices mat1 and mat2. If all elements are not different more than epsilon, the matrices are considered as equals, and null is returned. Otherwise, a String of the form "i, k, mat1k, mat2k" is returned showing the first encountered unequal matrix element.
