Classes

LMfit
A general curve fitter using an implementation of Levenberg-Marquardt, based on https://github.com/reptillicus/jsfit. Usage: 1) call constructor, 2) call fit() NOTE: On http://statpages.info/nonlin.html you can fit an arbitrary function to arbitrary values, can be used for testing.

Typedefs

FitFunction(double x, List<double> pars) → double
Fit function definition to be used as the fit model by LMfit. This is the function to which the data will be fitted ("model function"). Will be called by LMfit during its work. Computes the model function value at position x from the current iterated model parameters pars. At the beginning pars are the initial parameters.
